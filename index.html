JsonTr JSON-based declarative transformation language for JSON (like XSLT for XML)

Source JSON JsonTr template Transformation result 
{
    "sup": {
        "sub": {
            "name": "middle"
        }
    }
}

{
    "~match /sup": [
        "before",
        "~value-of sub/name",
        "after"
    ]
}

[
    "before",
    "middle",
    "after"
]
